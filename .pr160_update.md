## Summary

## Why
Issue #157 noted that representing a two-value flag as an enum adds unnecessary complexity. Using a boolean improves readability and removes conversion/match logic.

## Behavior / Compatibility
- CLI usage remains: `--daemon-mode true|false`
- Default behavior unchanged: daemon mode is still the default when `daemon_mode` is unset.
